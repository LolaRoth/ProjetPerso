Placez ici vos fichiers de polices personnalisées (.woff, .woff2, etc.).

Recommandation :

- Fournissez au moins le format `.woff2` pour de meilleures performances.
- Nommez les fichiers de façon explicite, ex : `MyCustomDisplay-Regular.woff2`.

Exemple :

- assets/fonts/MyCustomDisplay-Regular.woff2
- assets/fonts/MyCustomDisplay-Bold.woff2

Dans `assets/css/tailwind.css`, un exemple `@font-face` est déjà présent. Ajustez le `src` pour correspondre aux fichiers déposés ici.
